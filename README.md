#🏝️ Treasure Island Game (Python)

A fun, beginner-friendly text-based adventure game built using Python. Your mission is to navigate through choices and find the hidden treasure 🏆.

🎮 Game Overview

Welcome to Treasure Island!
You are placed in a mysterious land where every decision you make determines your fate.

Make the right choices… or face Game Over 💀

🧾 Game Flow
Start at a crossroad
Choose direction:
"left" or "right"
Next decision:
"wait" or "swim"
Final choice:
"red", "blue", or "yellow"

🧠 Game Logic Summary
Only one correct path:
Left → Wait → Yellow → 🎉 WIN

All other paths lead to Game Over

❌ Game Over Scenarios
Wrong direction → fall into a hole
Swim → attacked by trout 🐟
Red door → burned by fire 🔥
Blue door → eaten by beasts 🐺
Invalid input → Game Over

🏆 Winning Condition
Choose:
Left → Wait → Yellow

💻 Sample Gameplay
Welcome to Treasure Island.
Your mission is to find the treasure.

You're at a crossroad. Left or Right: left
Wait or Swim: wait
Which door? Red, Blue, Yellow: yellow

You WIN 🎉
